# Romantic Website

This is a simple one-page website you can open directly in a browser.

## Customize it

1. Open `script.js`.
2. Edit the values in the `siteContent` object near the top.
3. Replace placeholder names, the anniversary date, reasons, memories, and promises.

## Run it

You can either:

- double-click `index.html`, or
- serve the folder locally with a simple static server if you prefer.
